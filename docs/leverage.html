6. なぜ「恐怖でレバ」は失敗し「強欲でレバ」は成功するのか

メカニズムの解説

恐怖ゾーン（F&G < 20〜30）の問題:

  • 勝率は70%前後で一見高いが、負けるときの損失が-20%〜-30%
  • レバ3xだと損失は-60%〜-90%。複利では一度の大損で元本が壊滅
  • 恐怖局面は高ボラティリティ → レバETFの減価も大きい
  • 「落ちるナイフ」を3倍の力で掴むことになる

強欲ゾーン（F&G >= 80）の優位性:

  • 勝率76%で、負けても最大-4.6%（SPY）
  • 3xレバでも最大損失は-14%程度で回復可能
  • 強欲局面は低ボラティリティ → レバETFの減価が小さい
  • 市場の全指標がポジティブ = モメンタムが崩れにくい

数式的な直感:

複利の世界では「期待値 = 平均リターン - 分散/2」（ケリー基準の近似）。 恐怖ゾーンは平均リターンは高いが分散が巨大で、レバをかけると分散項が急増し期待値がマイナスに転落する。 強欲ゾーンは分散が小さいため、レバをかけても期待値が維持される。

7. 実用的な提言

100万円規模の個人投資家向け

推奨戦略: 「強欲ブースト」

  1. 通常時: SPY（またはeMAXIS Slim S&P500）をバイ&ホールド
  2. F&G >= 70〜80 のとき: ポジションの一部をSSO（2x）またはUPRO（3x）に切り替え
  3. F&G < 30 のとき: レバは絶対にかけない。通常の1xで追加投資するのみ

やってはいけないこと:

  • 恐怖ゾーンでのレバレッジ（「暴落で3倍買い」は破産への道）
  • F&G単独でのフル切り替え（F&Gは参考指標であり、売買シグナルではない）

期待される効果:

  • SPY BH対比でSharpe改善（0.76 → 1.0前後）
  • MDD据え置き（-32%程度）でCAGR向上（+12% → +17〜18%）
  • 恐怖局面でのダメージを増幅させない安全設計
