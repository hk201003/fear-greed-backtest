3. 記事の主要主張の検証

主張1: 「20-40レンジ + 21日保有」が最優秀

部分的に支持
PF 2.07〜3.16で確かに良好。ただし21日保有のPFランキングでは90-100（PF 5.56, N=10）や 80-90（PF 2.23）も上位。サンプルが十分な中では20-40が優秀、という主張は妥当。

主張2: 「60-70レンジ」は罠

条件付きで支持
2日保有ではPF 0.97と確かに悪い。ただし21日保有ではPF 1.82, 勝率65.5%と普通にプラス。 「短期では罠、長期では問題ない」が正確。記事は短期データだけで「罠」と断定しており誇張気味。

主張3: 「70-80 x 5日」モメンタム有効

支持される
PF 2.23, 勝率64.5%。記事のPF 2.32と近い値。短期モメンタム効果は再現された。

主張4: 「90-100 x 21日」PF 4.77（少数サンプル）

数値は近いがサンプル不足
検証でもPF 5.56（N=10）と高いが、10件では統計的に信頼できない。

8. 総合結論

記事の検証結果

  • 数値の正確性: 93%一致 — 記事のバックテスト結果はほぼ正しい
  • 主張の妥当性: 部分的 — 数字は正しいが解釈に誇張がある

記事が見落としている重要な点

  1. ベンチマーク比較の欠如: SPYは21日持てばPF 1.9。F&G戦略の超過リターンは+0.5%程度
  2. データマイニング: 80通り（10デシル × 8保有期間）から最良を選んでいる
  3. CAGR換算するとバイ&ホールドに負ける: どのF&G戦略も市場離脱期間のリターンを逃す
  4. 最悪ケースの軽視: 恐怖ゾーンで買った場合の最大損失は-20%超

実用的な結論

F&Gは「売買シグナル」ではなく「資金配分の参考指標」として使うのが妥当。

  • 主力はSPYのバイ&ホールド（または定期積立）
  • F&G < 20 → 追加資金を投入するタイミングの目安
  • F&G > 80 → 新規投入を控える（売る必要はない）
  • F&Gだけで「買い/売り」を判断するのは非効率
