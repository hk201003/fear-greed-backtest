Part 2: レバレッジ戦略

勝率の高いゾーンにレバレッジをかけたらどうなるか？ SSO(2x)・UPRO(3x)の実データを使った13戦略の比較検証。

恐怖レバ3x → CAGR -1.4%

主要な発見

  • 記事の数値はほぼ正確 — 24項目中22項目が一致（93%）
  • ただしベンチマーク比較が欠如 — SPYを21日持てばPF 1.9。F&G戦略の超過リターンは+0.5%程度
  • 恐怖ゾーンでのレバは最悪手 — 勝率70%でも最大損失-20%。3xで-60%、事実上の破産
  • 強欲ゾーンでのレバが正解 — F&G>=80は最大損失-4.6%。3xでも-14%で回復可能
  • 最高効率: 強欲2x+通常1x — Sharpe 1.04、CAGR +18.4%、MDD -32%（SPY BHと同水準）
  • どのF&G戦略もバイ&ホールドに勝てない — 市場離脱期間のリターンを逃すため

データ・手法

  • F&Gデータ: GitHub CSV (2011-2023) + CNN API (2024-2026) = 3,797件
  • 価格データ: SPY/SSO/UPRO (yfinance)
  • 期間: 2011-01-03 ～ 2026-03-06（約15年）
  • 執行: シグナル翌営業日のOpen → N営業日後のOpen
  • 方式: 非オーバーラップ（記事と同一条件）
